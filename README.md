网易严选仿站
===========
基于原生JS,仿PC端首页与商详页
---------------------------

主要功能
--------
首页导航与吸顶导航，样式变换

面向对象轮播图组件

图片懒加载

图片鼠标移入缩放效果

各类选项卡

根据产品id解析URL，以及返回404页面判定

购买商品选择规格功能（数据交互）

限时购倒计时

评论数据展示，有无图类别筛选

评论翻页模块

localStorage实现购物车

兼容性
-----
IE10以上，IE9部分兼容（不支持弹性布局—）
